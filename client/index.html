About Me

Hello, and welcome to my site. I built this website partly as a CV and partly to update projects I am working on, including GIS/cartography, web apps, homebrewing, paranormal investigation, and other random stuff. This site was built using a Bootstrap template and custom javascript to add functionality. It sits on a simple NodeJS framework using an Express web server and is deployed on the Heroku PaaS.
